# dotfiles

These are my dotfiles for linux/macos.

## Requirements
- kitty -> terminal
- neovim -> editor
- ripgrep -> for fast searching
- tmux -> for terminal window management

## How to install

- Download or clone this repository
- Run ./install to install the dotfiles
